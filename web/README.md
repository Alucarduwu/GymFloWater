# 🌐 GymFlow Web 🐟

Bienvenido a **GymFlow Web**, el MVP original del ecosistema **GymFlow**. Esta aplicación web reactiva permite registrar y visualizar entrenamientos desde cualquier navegador de forma rápida y sencilla.

---

## ✨ Características de la Versión Web

- **⚡ Desarrollo Rápido**: Construido sobre **Vite** para una experiencia de desarrollo instantánea.
- **🎨 UI Moderna**: Estética "Deep Sea" con Glassmorphism y Tailwind CSS.
- **📊 Gráficas de Progreso**: Visualización de volumen semanal.
- **📱 Responsive**: Diseño adaptado para dispositivos móviles a través de la web.
- **🔒 LocalStorage**: Almacenamiento rápido en el navegador para pruebas directas.

---

## 🛠️ Stack Tecnológico

- **🚀 Framework**: [React 19](https://react.dev/) - La base de una UI reactiva moderna.
- **⚡ Bundler**: [Vite](https://vitejs.dev/) - Cargas instantáneas y performance optimizado.
- **🎨 Styling**: [Tailwind CSS 4.x](https://tailwindcss.com/) - Diseño basado en utilidades de vanguardia.
- **📦 Iconografía**: [Lucide React](https://lucide.dev/) - Iconos claros y estilizados.
- **✨ Animaciones**: [Framer Motion / Motion for React](https://motion.dev/) - Transiciones suaves.

---

## 🚀 Cómo Empezar

### 1. Requisitos Previos
- Node.js (v18+) instalado.
- npm o pnpm.

### 2. Instalación de Dependencias
Ejecuta el siguiente comando en la raíz de la carpeta `web`:
```bash
npm install
```

### 3. Ejecutar en Modo Desarrollo
Inicia el servidor local:
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`.
