> **"Tu progreso no es lineal, pero tu constancia debe serlo."**

**GymFlow** es la solución definitiva para el atleta moderno que valora la privacidad, la rapidez y el diseño. Olvida los registros complicados; GymFlow está diseñado para que pases menos tiempo en el móvil y más tiempo bajo la barra.

> **Domina tus entrenamientos, conquista el océano del fitness.**

**GymFlow** no es solo una aplicación de seguimiento de gimnasio; es un ecosistema diseñado para potenciar tu progreso físico mediante visualización de datos, gamificación y una experiencia de usuario de primer nivel.

---

## 🏗️ Estructura del Proyecto

Este monorepo contiene dos implementaciones del sistema GymFlow:

### 📱 [GymFlow Mobile (Recomendado)](./mobile)
**Nuestra joya de la corona.** Una aplicación nativa construida con **Flutter** para una experiencia fluida, animaciones premium y almacenamiento local ultra-rápido.
- **🚀 Estado**: Estable / Producción local.
- **🛠️ Tech**: Flutter, SQLite, Provider, fl_chart.
- **📦 Distribución**: Android (.apk) e iOS.

### 🌐 [GymFlow Web](./web)
El proyecto original. Una aplicación web reactiva diseñada para una entrada rápida de datos y visualización analítica.
- **🛠️ Tech**: React 19, Vite, Tailwind CSS, Lucide.
- **🚀 Estado**: MVP / Legacy.

---

## 📱 GymFlow Mobile: La Joya de la Corona

La aplicación móvil es nuestra implementación principal. Ofrece:
- **📊 Power Analytics**: Visualización en tiempo real de tu volumen por sesión y semana.
- **🛡️ 100% Offline**: Tus datos nunca salen de tu dispositivo. Sin cuentas, sin rastreo.
- **🐠 Sistema de Rangos**: Evoluciona desde Huevo de Pez hasta Pez Espada según tu racha de entrenamientos.
