## 🧩 Description | Descripción

**EN 🇺🇸**  
GymFlow is a high-performance training system designed to track **strength, volume, and progression** using an **offline-first architecture**.  

It focuses on minimizing friction in data entry while maintaining deep analytical capabilities.

**ES 🇲🇽**  
GymFlow es un sistema de entrenamiento de alto rendimiento diseñado para registrar **fuerza, volumen y progreso** utilizando una arquitectura **local-first**.  

Se enfoca en eliminar fricción en el registro de datos sin perder profundidad analítica.

---

## 🎯 Problem | Problema

**EN 🇺🇸**  
Most fitness apps:
- are slow or depend on internet  
- lack real training analytics  
- focus only on logging, not performance  

**ES 🇲🇽**  
La mayoría de apps fitness:
- son lentas o dependen de internet  
- no tienen análisis real de entrenamiento  
- solo registran datos sin interpretar rendimiento  

---

## 💡 Solution | Solución

**EN 🇺🇸**  
GymFlow introduces a system focused on:

👉 **real performance tracking + zero-latency interaction**

Using local storage and reactive architecture for instant feedback.

**ES 🇲🇽**  
GymFlow introduce un sistema enfocado en:

👉 **seguimiento real de rendimiento + interacción sin latencia**

Utilizando almacenamiento local y arquitectura reactiva.
